# RandGen Web Application

Простий веб-додаток на Python, що генерує випадкові числа та демонструє базову роботу веб-сервісу.

Проєкт використовується для демонстрації практик **CI/CD**, **SAST аналізу** та роботи з **GitHub Actions** у межах лабораторної роботи.

---

# Функціональність

Додаток надає:

- генерацію випадкових значень
- простий веб-інтерфейс
- endpoint для перевірки стану сервісу (`/healthz`)
